# SINTAXE DO CSS

````css
    p{
        color: red;
    }
````
Sintaxe:

    seletor{
        propriedade: valor;
    }


- posso definir varias propriedades;

````css
    p{
        color: red;
        font-size:14px;
    }
````

- posso definir vários seletores (seletores agrupados(por virgula));

````css
    h1, h2, p{
        color: red;
        font-size:14px;
    }
````

- posso definir vários seletores (seletores ENCADEADOS(espaço));

````css
    div p strong a{
        color: red;
    }
````
